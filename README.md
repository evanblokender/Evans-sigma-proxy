# Evan's Sigma Proxy 🔥

Browser-style proxy powered by **Scramjet**. Deploy in one click to Render.

## Deploy to Render

1. Push this folder to a **GitHub repo**
2. Go to [render.com](https://render.com) → **New Web Service**
3. Connect your GitHub repo
4. Render auto-detects `render.yaml` — hit **Deploy**
5. Done. Your proxy is live ✓

## Features
- 🗂️ Multi-tab browser UI (Ctrl+T, Ctrl+W, Ctrl+L)
- 🔍 Address bar with search + URL support
- ⬅️ Back / Forward / Reload / Home buttons
- 🚀 Scramjet + Wisp for fast proxying
- 🎮 Quick links: YouTube, Discord, CoolMath, Roblox & more

## Local Dev
```bash
npm install
npm start
# Open http://localhost:8080
```
